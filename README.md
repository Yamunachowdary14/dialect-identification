# Dialect Identification for Indic Text

## 📌 Project Description
This project identifies Telugu dialects using AI and slang-based input.

## 🚀 Features
- Dialect prediction using NLP
- Supports 10 Telugu regions
- Contact form with database storage
- Interactive UI

## 🛠 Tech Stack
- Python (Flask)
- Transformers (HuggingFace)
- HTML, CSS, JavaScript
- SQLite

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
