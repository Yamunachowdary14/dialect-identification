About the Project

This project is an AI-based system that identifies Telugu dialects using slang sentences. Telugu language has many regional variations, and each region uses different words and tones in daily conversations.

📍 Telugu Dialect Areas & Slangs

  • Telangana: Uses words like "baava", "ra", "endi"
  • Guntur: Rough and fast slang like "oy", "ra bey"
  • Krishna: Polite language using "andi", "meeru"
  • Godavari: Soft tone and respectful speech
  • Chittoor: Mixed influence, words like "dei", "nuv"
  • Srikakulam: Unique coastal slang like "bey"
  • Vizag: Modern slang with casual words like "bro"
  • Kurnool: Strong Rayalaseema slang like "rey"
  • Anantapur: Uses "macha", "raa" in daily speech

🤖 Why This Project?

This project helps in understanding regional language differences. It is useful for Natural Language Processing (NLP), chatbots, and language-based AI applications.
